BJP5 Self-Check 2.31: starExclamation1
What is the output of the following sequence of loops?

for (int i = 1; i <= 2; i++) {
    for (int j = 1; j <= 3; j++) {
        for (int k = 1; k <= 4; k++) {
            System.out.print("*");
        }
        System.out.print("!");
    }
    System.out.println();
}

output
****!****!****!
****!****!****!
